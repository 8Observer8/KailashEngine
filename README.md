KailashEngine 2.0

# **Keyboard:** #

* Movement = WASD CTRL Space
* Flashlight = F
* Reel in picked object = Z
* Throw picked object = X
* Mouse Lock = CAPS Lock

### Toggles ###
* Toggle Fly Mode = F9

# **Mouse** #

* Pick / move physical object / Move camera when mouse unlocked = Left Click
* Zoom camera = Right Click
